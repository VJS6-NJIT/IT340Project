# What Does the FOX GOD Say?

A BABYMETAL fan web application created for IT340.

## 🎯 Project Overview
This project is a full-stack web application that allows users to:
- View tour dates
- Browse merchandise
- Create and log into accounts

## 🧩 Features (Milestone 1)
- Home page (band showcase)
- Login and Register pages
- Tour page (UI)
- Merchandise page (UI)
- Navigation bar and footer
- BABYMETAL-themed design

## 💻 Technologies Used
- HTML
- CSS
- Git & GitHub
- Ubuntu VM (for hosting)

## 🚀 Status
Milestone 1 Complete  
Frontend deployed on VM and pushed to GitHub.

## 📌 Future Plans (Milestone 2 & 3)
- User authentication (Node.js + Express)
- MongoDB database
- API integration (tour lookup)
- Cart system
- Logging and security features
